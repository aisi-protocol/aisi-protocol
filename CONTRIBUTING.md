# Contributing to AISI Protocol

We welcome contributions to the protocol specification, tools, and documentation.

## How to Contribute
1.  **Report Issues**: Use the [GitHub Issues](https://github.com/aisi-protocol/aisi-protocol/issues) page for bugs, questions, or suggestions.
2.  **Propose Changes**:
    - Fork the repository.
    - Create a feature branch (`git checkout -b feat/your-idea`).
    - Make your changes, ensuring they align with the protocol's vision of clarity for agents.
    - Test your changes (e.g., run the validator on updated examples).
    - Submit a Pull Request (PR) with a clear description.
