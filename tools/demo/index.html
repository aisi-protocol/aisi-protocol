AISI Protocol 交互演示

展示AI智能体如何通过AISI协议发现和调用披萨外卖服务
协议版本: v1.0 (2026-01)

服务发现流程

1
AI获得服务标识符

用户说："我想订一份披萨" → AI智能体需要找到披萨外卖服务

用户意图: "我想订一份披萨"
服务标识符: aisi://did:web:pizzacompany.com/pizza-delivery

AI从知识库或搜索引擎获得AISI服务标识符

2
解析DID文档

AI解析DID标识符，查找对应的DID文档

GET https://pizzacompany.com/.well-known/did.json

{
  "@context": "https://www.w3.org/ns/did/v1",
  "id": "did:web:pizzacompany.com",
  "service": [
    {
      "id": "#aisi-pizza-delivery",
      "type": "AISIService",
      "serviceEndpoint": "/.well-known/aisi/pizza-delivery.json"
    }
  ]
}
3
获取AISI服务描述

AI从DID文档中找到AISI端点，获取完整的服务描述

GET https://pizzacompany.com/.well-known/aisi/pizza-delivery.json

点击"获取描述"查看完整的AISI服务描述
4
验证AISI格式

AI验证AISI描述文件是否符合协议规范

验证检查:
• aisi_protocol = "2026-01" ✓
• discovery.identifier 格式正确 ✓
• metadata 必填字段完整 ✓
• api.endpoint 有效 ✓
• JSON格式正确 ✓
5
调用披萨下单API

AI使用AISI描述中的信息调用实际的服务API

POST https://api.pizzacompany.com/v1/orders

{
  "pizza_type": "玛格丽特披萨",
  "size": "大号",
  "delivery_address": "北京市海淀区...",
  "payment_method": "alipay"
}
