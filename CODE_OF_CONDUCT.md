# 贡献者公约行为准则

## 我们的承诺

我们作为成员、贡献者和维护者承诺，使参与我们的社区成为一种不受骚扰的体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性身份和性取向如何。

我们承诺以有助于建立开放、友好、多样化、包容、健康社区的方式行事和互动。

## 我们的标准

有助于为我们的社区创造积极环境的行为包括：

* 对他人的不同观点和经验表现出同理心和善意
* 给予和优雅地接受建设性反馈
* 接受责任并向受我们错误影响的人道歉，并从经验中学习
* 专注于对社区最有利的事情，而不仅仅是个人利益

不可接受的行为包括：

* 使用性暗示语言或图像，以及任何形式的性关注或接近
* 挑衅、侮辱或贬损性评论，以及个人或政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息，例如物理地址或电子邮件地址
* 在专业环境中可能被合理认为不适当的其他行为

## 执行责任

社区维护者负责澄清和执行我们的行为标准，并将对任何他们认为不适当、具有威胁性、冒犯性或有害的行为采取适当和公平的纠正措施。

社区维护者有权和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki编辑、问题和其他贡献，并将在适当时沟通执行原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中代表社区时的行为。代表我们社区的例子包括使用官方电子邮件地址、通过官方社交媒体帐户发帖，或在在线或线下活动中担任指定代表。

## 执行

虐待、骚扰或其他不可接受行为可以通过 [aisi-conduct@googlegroups.com] 向负责执行的社区维护者报告。所有投诉都将得到及时和公正的审查和调查。

所有社区维护者都有义务尊重事件报告者的隐私和安全。

## 执行指南

社区维护者将遵循以下社区影响指南来明确违反本准则的后果：

### 1. 纠正
**社区影响**：使用不恰当的语言或其他在社区中被认为不专业或不受欢迎的行为。

**后果**：社区维护者的私人书面警告，明确违规性质，解释行为为何不适当。可能会要求公开道歉。

### 2. 警告
**社区影响**：通过单个事件或一系列行动违规。

**后果**：警告，指出持续行为的后果。在规定时间内不得与相关人员进行互动，包括与执行行为准则的人员进行主动互动。这包括避免在社区空间以及外部渠道（如社交媒体）中进行互动。违反这些条款可能会导致临时或永久禁令。

### 3. 临时封禁
**社区影响**：严重违反社区标准，包括持续不适当行为。

**后果**：在规定时间内暂时禁止任何形式的互动或公开与社区沟通。在此期间，不允许与相关人员进行公开或私下互动，包括与执行行为准则的人员进行主动互动。违反这些条款可能会导致永久禁令。

### 4. 永久封禁
**社区影响**：表现出违反社区标准的行为模式，包括持续不适当行为、骚扰个人或攻击或贬低某个类别的个人。

**后果**：永久禁止在社区内进行任何形式的公开互动。
