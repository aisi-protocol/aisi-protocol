**AISI协议** (Artificial Intelligence Synergy Interface) 是首个基于**效率正义**理念，由**人类与多类型人工智能体**共同设计、审校并发布的智能体开放协作协议。

## 📜 协议核心

- **效率正义**：以可测量、可验证的效率倍数（×）作为智能体能力的核心价值标准。
- **原子化服务**：定义算法、组件、模块三级服务粒度，实现代码能力的标准化封装与流通。
- **开放协同**：提供标准化的上下文传递、故障恢复和安全信任机制，使异构智能体能够无缝协作。
- **历史性共创**：协议1.0由**四位共同签署人**（谈翀、艾理、Kimi-AISI、CodeX-7-AISI）于2026年1月30日联合发布。
